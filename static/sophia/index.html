Colinha HTML

Estrutura de um arquivo HTML

Um arquivo HTML completo envolve desenas de tags, cada uma delas tem um propósito e um resultado especial.

A tag html envolve todas as outras tags exceto a tag especial "!DOCTYPE html". Dentro dela, deve conter as tags "head" e "body".

A tag head contém meta-informação sobre este arquivo html, como o título da página no topo do navegador, codificação (encoding) da página, ícone, ou até inclusão de arquivos.

A tag body contém todo o conteúdo da página: tags de parágrafos("p"), títulos e subtítulos ("h1", "h2",...), imagens("img"), etc.

A anatomia de uma tag

Neste exemplo, a tag p é tag de abertura, /p a tag de fechamento."Meu gatinho é muito mal humorado" o conteúdo da tag e todos eles compondo um elemento.

Uma tag precisa ser aberta, mas nao nescessariamente fechada.

Tags de abertura podem ter atributos:

img src="bob-esponja.png"

Em img, o atributo src aponta para a URL do arquivo

Atributos universais

Para selecionar apenas alguma parte específica e estiliza-la é possível usar os atributos universais HTML: class e id.

Até agora, estilizamos elementos HTML de duas formas:

Primeira forma: selecionando a tag:

p { color: blue; }

Segunda forma: selecionando um elemento em específico criando uma id para ele. Supondo que temos:

p id="resumo" Este é o resumo da notícia.../p

#resumo { color: blue; }

Como fazemos, então, para estilizar não apenas um, mas um subconjunto de elementos da forma como queremos? Usando classes

p class="destacado">Primeiro /p

p class="destacado">Segundo /p

E, em um arquivo CSS, podemos escrever o nome da tag, seguido por um ponto ".", seguido pelo nome da classe:

p.destacado { font-weight: bold; }

.destacado { font-weight: bold; }

Tags de conteúdo e seus atributos

Tag de Paragrafo (p)

Quebras de linha são feitas automaticamente.

Tags de Títulos e Subtítulos ( h1 . . . h6 )

Podemos ter até seis níveis de títulos! Mas a quantidade de títulos é infinita.

Tag de Imagem (img)

Para exibir imagens:

img src=". . ." alt=". . ."

O atributo src=". . ." (abreviação de source) é usado para definir o endereço (URL) da imagem. Este endereço pode ser externo ou interno

URL externa x interna

URL externa

Aponta para algo (imagem etc.) de um outro site. Exemplos:

http://google.com

http://fegemo.github.io/imgs/kraken.png

https://twitter.com/elonmusk

URL interna

Aponta para algo da própria página/site

Pode ser um caminho relativo ou absoluto:

racas-raras.html (relativo)

/arquivos/fotosdafesta.zip (absoluto)

Tag de Hyperlink (a href="".../a)

Link externo (para fora da página):

a href="http://www.google.com">Texto do link externo/a

Link interno (para algo hospedado no próprio computador):

a href="downloads/exemplo.zip">Texto do link interno/a

Como incluir código CSS

Um arquivo HTML podoe referenciar um CSS assim:

link rel="stylesheet" href="arquivo-de-estilos.css"

Assim mais de um arquivo HTML pode usar esse CSS, ou seja, o código CSS é reaproveitado

Como colocar um ícone

Para adcionar um ícone(favicon) à sua página HTML basta você utilizar a tag link.

link rel="icon" href="imgs/url-da-imagem"

Como o ícone não faz parte das informações que serão exibidas na página, ele deve ficar dentro da tag head e geralmente abaixo da tag title.
