Atributos de tag

Tags são linguagens de marcação para a estrutura que quer adicionar ao seu texto. Cada uma contém uma instrução para que a página web renderize, como um rótulo.

A tage de abertura inicia-se com "<" e acabam com ">", e a tag de fechamento inicia-se "", colocando seu conteúdo entre elas. Exemplo:

Exemplo: < p >...< /p > < mark >...< /mark >

Tags universais

Existe duas tags universais feitas para selecionar elementos especificos. O 1° é o "id" que seleciona um elemento específico, dando-lhe um nome de sua escolha, é ótimo para estilizar apenas tal tag e bom para um hyperlink da prórpia página.

PS: não se pode criar dois "id" com o mesmo nome, para isso crie com o próximo elemento.

O 2° é o "class" que tem a mesma função do "id", porém, seu diferencial é que seleciona um conjunto de tags da sua escolha, dando-os também facilidade para estilizar oque pertence a classe, sem codecisar dar um id para cada elemento.

Exemplo: < p id="exemplo" > e < p class="exemplo" >

Tags de agrupamento

Temos duas tags de agrupamento, "div" e "span" que servem para aglomeração, ambas não tem semântica porém são muito utilizadas para criar uma unificação, melhorando na estilização.

Exemplo: < div >...< /div > é block. Já < span >...< /span > é inline.

Como por um icone na aba?

Para adicionar um iconezinho na aba é facil, adicione < link rel="icon" href="..." >com o nome do arquivo nos 3 pontinhos.

PS: lembre-se de por uma foto com boa resolução em pequeno tamanho.

Como por um arquivo css no HTML?

Para adicionar um outro arquivo com todas propriedade css é a mesma lógica do icone, < link rel="stylesheet" href="..." > e no lugar do três pontinhos, coloque um arquivo .css nele e prontinho!

Elementos semânticos

Substitui algumas div/span, pois organiza e dá mais entendimento ao navegador.

Primeiro, os block.

  • < header >< /header >
    • Cabeçalho da página.
  • < main >< /main >
    • Conteúdo principal da página.
  • < footer >< /footer >
    • Rodapé da página.
  • < main >< /main >
    • Um componente completo da página, como um post em um site ou um produto em uma loja.
  • < section >< /section >
    • Uma seção lógica da página.
  • < nav >< /nav >
    • Uma seção feita para navegação, contendo então links da página.

Agora os inline

  • < aside >< /aside >
    • Uma seção periférica da página.
  • < time >< /time >
    • Recodesenta alguma data/horário codesente na página.
  • < mark >< /mark >
    • Recodesenta uma marcação de texto, estilo marcadores.
