O que é um atributo?

Atributo são informações que passamos na Tag para que ela se comporte da maneira esperada.

Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de epecificação).

Estrutura de um arquivo html

A estrutura de um arquivo html são um punhado de tags no texto. Essas tags fazem a marcacçaõ da estrutura do texto.

Existem dezenas de tags e cada uma delas tem um propósito e um resultado visual, sendo que a maioria delas precisam ser fechadas.

Atributos universais

class: Determina uma (ou mais) teclas de atalho para o elemento. Para definir mais de uma tecla, coloque-as separadas por espaço.

hidden: Oculta o elemento onde ele for declarado.

id: É o identificador único do elemento. Somente deve ser declarado um id por elemento. E este id não deve ser repetido na mesma página.

lang: Determina o idioma em que está escrito o conteúdo do elemento.

style: Determina propriedades css diretamente no elemento.

title: Representa um auxílio ao elemento. Semelhante a dica do elemento.

Para colocar um ícone (favicon) devemos indicar o caminho com o link nas tags do cabeçalho head.

Principais tags de conetúdo e seus atributos

html: Envolve todo um documento html.

head: Envolve o cabeçalho de um documento html.

meta: Fornece informações gerais sobre o documento.

style: Informações de estilo.

script: Linguagem script.

title: O título do documento.

body: Envolve o corpo (texto e tags) do documento html. background-color: Cor de fundo; background: Imagem como plano de fundo; color: Cor do texto principal; link: Cor dos links existentes na página; marginwidth: Elimina a margem no topo da página; marginheight: Elimina a margem esquerda; topmargin: Elimina a margem no topo da página; leftmargin: Elimina a margem esquerda.

p: Um simples páragrafo.

a: Cria um link e inclui atributos em comum. ref: O URL do documento que será vinculado a este. Para e-mail: mailto e link externo: http; name: O nome da âncora; target: Identifica a janela ou local em que o link deverá ser aberto: blank, self, top, parent; rel: Define os tipos de link que avançam.

ol: Uma lista ordenada.

ul: Uma lista não ordenada.

li: Um item da lista.

menu: Um menu com uma lista de itens.

em: Maior ênfase em itálico.

strong: Maior ênfase em negrito.

mark: Representa um trecho de destaque em um texto.

del: Serve para apagar um trecho.

div: É um agrupamento de elementos. É um elemento block.

span: É um agrupamento de elementos. É um elemento inline.

img: Insere uma imagem in-line no documento e inclui atributos comuns.

table: Cria uma tabela.

caption: A legenda para a tabela.

tr: Uma linha na tabela.

td: Usada em todas as outras linhas.

th: Uma célula do cabeçalho da tabela.

Como incluir código css

A tag link é uma tag vazia, assim como br e meta. O atributo href indica o endereço do arquivo css, como visto acima o arquivo se chama "arquivo.css" e está dentro de uma pasta "css". O atributo rel determina a relação do link com a página, sendo ela stylesheet ou folha de estilos.
