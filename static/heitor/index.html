introdução

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.

  • h2,h3{
  • ...color: darkred;

  • ...padding-left: 600px;
  • }

color,font-family,padding-left = é a declaração que contém a propriedade, e os respectivos valores são darkred,cursive,600px

cores

as cores num documento html ou xhtml, podem ser referentes à muitas partes do documento, e podem ser em hexadecimal, nome,ou rgba

em hexadecimal, as cores são descritas segundo uma tabela,numerada em hexadecimal

por nome, simplesmente, a cor é definida segundo seu nome na tabela

em rgba, seu nome é definido por rgb(número da tabela)

pode-se também usar duas ou mais cores usando o background-image: linear gradient(para onde,cor1,cor2,...corN)

as cores podem definir as partes de um documento como:

  • o fundo(background-color)
  • o texto(color)
  • a borda(border-color)
  • etc

Box model

o box model é basicamente o tamanho daquilo presente no html que pode ser alterado com o css.
