O html é usado para formar enfim uma página.

Informações Gerais

  1. Criador: Tim Berners-Lee
  2. Significado:HyperText Markup Language em português, linguagem de marcação de hipertexto.

    3. PS: hipertexto - Hipertexto é o termo que remete a um texto ao qual se agregam outros conjuntos de informação na forma de blocos de textos, palavras, imagens ou sons, cujo acesso se dá através de referências específicas, no meio digital denominadas hiperligações.

  3. Surgiu: O HTML surgiu em 1990

    4. Anatomia de uma TAG

    Tags são o conjunto de caracteres que formam um elemento

    E para uma tag funcionar não devem conter espaços

    E lembrando que a tag sempre se fecha com o /, ou seja Para as que necessitam de fechamento, utilizamos o sinal de menor ( < ), seguido do nome do elemento e o sinal de maior ( > ) para abertura. Para fechamento, utilizamos o sinal de menor ( < ), seguido de barra ( / ), nome do elemento e o sinal de maior ( > ).

    Um exemplo na tag: < img src= "" >

    ATRIBUTOS

    Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada.

    Atributos Universais:

    As classes são uma forma de identificar um grupo de elementos. Através delas, pode-se atribuir formatação a VÁRIOS elementos de uma vez. Exemplo:

    div class: "classe1"

    Depois de adcionar uma div class no seu html, você consiguirá editar os elementos, apenas selecionando aquela classe, pois ela vai estar redirecionada aos elementos.

    As ids são uma forma de identificar um elemento, e devem ser ÚNICAS para cada elemento.

    Ao acrescentar um id no seu html, poderá seleciona-lo indicando o id em que tal está redirecionado

    Vemos assim que a diferença é que um id é único e o class é mais geral

    TAGS MAIS USADAS
    • A declaração "< !DOCTYPE>" deve ser a primeira coisa no seu documento HTML, antes da tag " < html >".
    • A tag < a > define um hiperlink, que é usado para vincular de uma página para outra.
    • A tag < article > especifica conteúdo independente, separando então em artigos
    • A tag < aside > define algum conteúdo além do conteúdo principal
    • A tag < audio > serve para colocar um audio na página ;)
    • A tag < body > define o corpo do documento.
    • A tag < caption > define uma legenda da tabela.
    • A tag < del > define o texto que foi excluído de um documento.
    • A tag < div > define uma divisão ou seção em um documento HTML.
    • A tag < em > serve para enfatizar o texto
    • A marca < font > especifica a face da fonte, o tamanho da fonte e a cor do texto. Temos então alguns tipos (muito USADAS dentro do CSS- estiliza texto)
      1. font family = estilo da letra
      2. font size = tamanho
      3. font color = cor
        4.
    • A tag < footer > define um rodapé para um documento ou seção
    • As tags < h1 > a < h6 > são usadas para definir títulos HTML.
    • O elemento < head > é um contêiner para todos os elementos principais.
    • A tag < ul > define uma lista não ordenada (com marcadores), e < ol> uma lista ordenada. Ambas usam a tag < li > para representar um item da lista
    • A tag < title > é necessária em todos os documentos HTML e define o título do documento.
    • A tag < strong > deixa o texto em negrito
    • A tag < section > define seções em um documento, como capítulos, cabeçalhos, rodapés ou quaisquer outras seções do documento.
    • A tag < p > define um parágrafo.
    • A tag < nav > define um conjunto de links de navegação.
    • A tag < mark > define o texto marcado.
    • A tag < img > define uma imagem em uma página HTML.
    • A tag < table > determina a criação de uma tabela
    • A tag < tbody > define o corpo da tabela
    • A tag define o rodapé da tabela
    • A tag < td > as células da tabela
    • A tag < th > representa a célula de cabeçalho de uma tabela
    • A tag < thead > representa o agrupamento do conteúdo de uma tabela
    • A tag serve para acrescentar links

    Adicionando FAVICON

    Para adicionar um favicon na página, isto é, aquela imagem que fica ao lado do título da página, deveremos usar a tag:<\p>

    link rel="shortcut icon" href= "'lugar que colocamos o link da imagem'.ico" >" .

    Um detalhe importante é que geralmente colocamos ele na parte de cima do nosso cod.html.

    COLOCANDO O CSS

    Para colocar o cod que estilize a página, a tag usada deverá ser

    < link rel="stylesheet" href="arquivo-de-estilos.css" >
