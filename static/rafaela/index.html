Os 101 dálmatas estilos

Há 5 variações:
[a] existência do atributo: [atributo]
Neste caso, todo < a > que possua href
[b] valor inteiro: [atrib="valor"]
[c] começo do valor: [atrib^="inicio"]
[d] fim do valor: [atrib$="final"]
[e] trecho do valor: [atrib*="trecho"]

Seletores de estado

a:link {
color: blue; /* [a] */
}
a:hover {
color: cyan; /* [b] */
}
a:active {
color: gold; /* [c] */
}

Há pelo menos 3 estados:
[a] situação inicial :link
[b] mouse em cima: :hover
[c] começo do valor: :active
Os seletores :hover e :active podem ser usados com outros elementos.

Negação

Formato: X:not(seletor)

p:not(.destacado) {
color: gray;
}

Descrição: seleciona todos os elementos que não passem pelo teste do seletor ou seja, no exemplo, seleciona todos os que não possuam a classe .destacado

Outros seletores

Seletores de posição dentro do pai:
X:first-child
X:last-child
X:nth-child(n)
Seletores de posição de um tipo:
X:first-of-type
X:last-of-type
X:nth-of-type(n)

A parte dos seletores “com dois pontos” (eg, :not(), :hover) é chamada de pseudoclasse.

Gradientes

body{
background-image: linear-gradient(to bottom right purple pink blue);
}
OU
body{
background-image: radial-gradient(purple pink blue);
}

Cores

As cores podem ser mostradas como seu nome, por exemplo Green, Red, Pink, pelo seu código hexadecimal composto por 6 caracteres sendo eles números ou letras, por exemplo #8B0000 e #A52A2A ou pelo seu código rgba sendo composto por 3 números separados por vírgulas, por exemplo (128,0,0) e (139,0,0).

Espaçamentos e tamanho

Margin e padding

p{
border:1px dotted black;
padding: 40px;
margin: 10px;
text-align:justify;
}

Padding: Espaçamento interno, da borda para dentro
Margem: Espaçamento externo, da borda para fora

Width e height

Podemos especificar a largura dos elementos block por meio da propriedade width

p {
width: 260px;
}

Já para especificar a altura de um elemento utilizamos a propriedade height

img {
height: 400px;
}

Propriedades de texto

font-size:Define o tamanho da fonte, e.g., 18px
font-weight:Define a espessura da fonte
Valores: normal, lighter, bold, bolder ou um número representando sua espessura
font-style:Define o estilo da fonte
Valores: normal e italic
text-decoration:Sublinha, risca ou coloca um risco acima do texto:
Valores: none (nenhum), underline (sublinhado), overline (acima do texto), line-through (riscado)

Bordas

A propriedade border é um atalho: border-width, border-style e border-color
Exemplo (os dois são equivalentes):

p {
border-width: 1px; /* largura de 1 pixel */
border-style: solid; /* borda toda colorida */
border-color: red; /* cor vermelha */
}
OU
p { /* preferimos esta forma, que é mais sucinta */
border: 1px solid red;
}

Box Model

O navegador enxerga todo elemento de conteúdo como uma “caixa”
A “caixa” é formada por:
-Espaço do conteúdo
-Espaço de preenchimento (padding)
-Bordas (border)
-Espaço externo (margin)

É possível alterar o significado da width e height de um elemento usando a propriedade box-sizing:
box-sizing: content-box (valor padrão)
width = largura do conteúdo
box-sizing: border-box
width = conteúdo + padding + border

Anatomia de uma regra CSS

Regra

h1{
text-align: center;
color: red;
font-family: Arial
}

Dentro de uma tag CSS existem seletores, declarações, propriedades e valores.Na regra apresentada acima o termo h1 seria o seletor, "text-align: center;" é um exemplo de declaração, onde "text-align" é a propriedade e "center" é o valor.

Principais tipos de seletores

Tag:

p {/* usamos 'p', que é o nome da tag de parágrafo */
margin-left: 5%;
}

Id:

#dálmata {/* usamos '#dálmata', que é o nome do id */
width:100px;
}

Classe:

.dálmata {/* usamos '.dálmata', que é o nome da classse */
width:100px;
}

Seletores mais avançados

Descendente

Formato: X Y (antecessor, espaço, elemento selecionado) Exemplo:

ul a {
text-decoration: none;
}

Descrição: seleciona todos hiperlinks a que têm um ul como antecedente (pai, avô, bisavô etc.)

Filho direto

Formato: X > Y (pai, sinal de maior, elemento selecionado) Exemplo:

#menu-principal > ul {
padding: 20px;
}

Descrição: seleciona todos os ul que têm #menu-principal como pai

Seletores de Atributo

Podemos selecionar elementos HTML de acordo com seus atributos:

a[href] {
color: blue; /* [a] */
}
a[href="http://google.com"] {
color: blue; /* [b] */
}
a[href^="http"] {
color: blue; /* [c] */
}
a[href$=".com"] {
color: blue; /* [d] */
}
a[href*="google"] {
color: blue; /* [e] */
}
