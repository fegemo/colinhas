O HTML é uma linguagem baseada em marcação.Essas marcações são chamadas de TAGS (as tags são abertas e depois fechadas). Utilizando as tags, nós dizemos para o navegador o que é cada informação. A sigla HTML vem do termo Hypertext Markup Language (Linguagem de marcação de hipertexto) e foi criador por Tim Berners-Lee em 1991.
