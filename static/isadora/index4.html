Id

As ids são utilizadas para identificar um único elemento com características próprias.

Class

Diferente das ids, as class são utilizadas para identificar um grupo de elementos com características semelhantes.

Para o que são utilizadas ?

As ids e as class são utilizadas quando queremos especificar um atributo para um único ou vários elementos ao mesmo tempo, sem atribuir a todos da sua mesma tag.
