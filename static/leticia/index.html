Estrutura do HTML

O arquivo HTML é formado por tags por principais:

Tags O que são
<!DOCTYPE html> Serve para especificar qual a versão html usada
<html> </html> É aquele que envolve todo o arquivo
<head> </head> Cabeçalho do arquivo;
<title> </title> Título da página;
<body> </body> Contém todo conteúdo do arquivo, os textos e tags

Anatomia das Tags

Um arquivo html é formado por diversas tags, a anatomia dela é composta por abertura, fechamento, conteúdo, sendo:

Parte da tag Função
<h1> Abre
<h1> Colas </h1> Conteúdo
</h1> Fecha
Também temos tags que permitem atribuição:
  • <img src="colas-para-prova.png">

O que são atributos?

Atributos são comandos ou informações específicas dadas para determinada tag, esse comandos são direcionados ao navegador para que ele interprete e apresente o que há no atributo. São recebidas pelo navegador assim: <img src="http://www.ligacaoteen.com.br/wp-content/uploads/2012/05/tec.ninj-de-colar-7.jpg"/> e são enviadas para a página assim :

Atributos Universais

Os atributos universais servem para selecionar dentro do código HTML um determinado elemento ou vários elementos, são chamados assim pois qualquer elemento pode receber.

  • Os ids são usados para estilizar elementos em css, para determinar seus nomes é só colocar depois do seletor " id="alguma coisa" " detalhe importante o id deve ser único no código e são iniciados com # no css;
  • As classes servem para por exemplo estilizar um subconjunto de elementos, para determinar um nome a classe é só colocar após o seletor " class="importantes" ", como os ids as classes também devem ser únicos no código e são iniciados com um ".".

Vamos as tags principais de um arquivo HTML

Tags principais

  • O parágrafo("<p> </p>"), e suas quebras de linhas são feitas automaticamente;
  • Os títulos e subtítulos("<h1> </h1>";) e os números após o "h" vai até o 6, podendo conter então 6 tipos de subtítulo;
  • A imagem("<img src="aqui-url-da-imagem">") quando estiver dentro de outro site e quando tiver dentro do próprio arquivo usa "<img src="endereço-relativo-ou-absoluto">";
  • Hiperlinks para links externos "<a href="http://www.google.com">Texto do link externo</a>", para links internos "<a href="downloads/exemplo.zip">Texto do link interno</a>" essa tag apresenta um atributo target que pode ter valor de abrir o link na mesma aba chamado _self e o _blank que abre o link em uma nova aba;

Tags de Importância

  • O Strong("<strong> </strong>") é usada para marcar textos importantes e são escritas em negrito;
  • O em("<em> </em>") são usadas para dar ênfase ao textos que são escritos em italico;
  • O del("<del> </del>") indicam que os elementos do texto foram removidos e riscados;
  • O ins("<ins> </ins>") sublinham os textos desejados;
  • O mark("<mark> </mark>") realça parte do texto, que pode ser importante para o leitor;

Elementos inline e block

Elementos inline e block são a forma que os códigos assumem na página, os elementos block são aqueles que iniciam uma nova linha e ocupam todo o espaço horizontal como o parágrafo os títulos, listas não ordenadas, já os inline ocupam apenas o espaço necessário e não precisam iniciar uma nvoa linha, como strong, em, mark.

Criando tabelas

Uma tabela é formada por linhas, colunas, sendo que uma é para o Cabeçalhoe e a última para o rodapé. Tabelas também contém legendas.

Tabelas Simples

As tabelas simples são formadas pelas tags:

  1. <table>...</table>
  2. <tr>...</tr>, linha da tabela
  3. <td>...</td>, célula de dados
  4. <th>...</th>, célula de cabeçalho

A função de cada tag da tabela

  • A tabela começa com a tag <table> e finaliza com </table>;
  • A cada linha utiliza <tr> para começar e </tr> para terminar;
  • <td> armazena os dados da tabela e a tag <th> armazena os dados do cabeçalho;
  • <td> e <th> devem estar dentro de uma linha (<tr>);
  • As tags <th> estão sempre em negrito.
Para mesclar células na horizontal usa se colspan="X", sendo que x é o número de colunas que deseja juntar e para vertical usa rowspan="Y", sendo y é o número de linhas que a célula vai conter. Esses valores são colocados após a tag mas dentro da ">".

Agrupamento de elementos HTML

Div

Div(<div></div>) serve para junção de elementos html, sozinho em um código ele não apresenta semântica, ou seja, não significa nada sozinho. É um elemento block. Agrupa por exemplo parágrafos, títulos e subtítulo, imagem.

Span

Span(<span></span>) apresenta mesma função que a Div, porém é inline.

Box Model

Box Model é a forma que o navegador vê todo o conteúdo dentro de uma "caixinha". Essa "caixinha" é formada por:

  • Espaço do conteúdo
  • Espaço de preenchimento (padding)
  • Bordas (border)
  • Espaço externo (margin)

Nas ordens dos itens acima começando de dentro para fora na imagem. E os números que aparecem são os tamanhos que cada elemento ocupa.

Quando se define valores largura e altura, você está definindo o tamanho do conteúdo da caixinha e não ela toda. É possível alterar o significado da width(largura) e height(altura) de um elemento usando box-sizing, uma propriedade:

  • box-sizing: content-box (valor padrão)
  • width = largura do conteúdo
  • box-sizing: border-box
  • width = conteúdo + padding + border
