HTML

E suas propriedades

Curiosidade!

O HTML foi criado pelo físico Tim Berners-Lee em 1990. Tim também criou o protocolo http, um servidor web e o primeiro navegador. Tudo isso porque ele percebeu a necessidade dos físicos de compartilhar informação e a falta de máquinas ou formatos comuns.

HTML é uma linguagem usada para desenvolver websites e é encarregada do conteúdo e da estrutura da página. A sigla HTML vem do inglês e significa Hypertext Markup Language ou, em português, Linguagem de Marcação de Hipertexto. Nessa página, iremos explorar as tags, estruturas, atributos e outras propriedades do HTML.

Estrutura de um arquivo HTML

Um arquivo HTML é constituído de tags. A primeira é a tag html, que engloba todas as outras tags (à exceção da tag especial !DOCTYPE). Dentro dela há duas outras tags, a head e a body, nessa ordem. A tag head apresenta toda a meta-informação da página, isto é, título da página no topo do navegador, codificação, ícone e inclusão de arquivos, por exemplo. Já a tag body apresenta todo o conteúdo da página, como títulos, parágrafos, imagens, entre outros.

Anatomia de uma tag

Uma tag html é composta pelos seguintes elementos:

  • Tag de abertura: como próprio nome diz, abre a tag;
  • Conteúdo: encontra-se entre a tag de abertura e de fechamento, é o que será exibido na página;
  • Tag de fechamento: fecha a tag;
  • Atributo: presente na tag de abertura e atribui algo à tag.

Tag de parágrafo

Em uma tag p (parágrafo) as quebras de linha são feitas automaticamente, ou seja, p é um elemento block.

Tag de Imagem

A tag de imagem, img, apresenta um atributo, o src, no qual se coloca a URL da imagem, ou seja, o endereço da imagem. Diferente da maioria das tags, a de imagem só apresenta a tag de abertura, não possuindo conteúdo ou tag de fechamento.

Tag de Hiperlink

A tag de hiperlink, a, apresenta o atributo href, no qual se coloca o caminho do link, seja ele interno ou externo. A tag de hiperlink apresenta conteúdo e tag de fechamento.

Tag de Lista

Listas podem ser numeradas, ol, ou não numeradas, ul. Em ambos os casos, cada item da lista está dentro de uma tag li.

Tags de Importâcia

As tags de importâcia são usadas para dar destaque ao texto. São elas:

  • Strong: marca textos com alta importância;
  • Em: dão ênfase ao texto;
  • Del: marca textos que foram eliminados;
  • Ins: marca um texto inserido.
  • Mark: realça um texto que é considerado importante ao leitor.

Elementos Box e Inline

O navegador tem diferentes formas de dispor os elementos. Há dois tipos principais de elementos, quando se trata de disposição, elementos box e inline.

Elementos box (ex.: p, h1...h6, ul, ol, li) são dispostos um abaixo do outro, tendo quebra de linha. Já elementos inline (ex.: img, a, strong, em, del, etc) são dispostos a direita um do outro, na mesma linha.

Atributos Universais

Existem atributos universais, isto é, que podem ser usados em qualquer tag. Esses atributos são usados para identificar um elemento ou um grupo de elementos. São esses id e class. O atributo id é usado para identificar um só elemento, ou seja, não pode haver um mesmo id em uma página. Já o atributo class é usado para identificar um mesmo grupo de elementos. Ambos são usados como seletores em um CSS.
