Anatomia de uma Tag

<p>Olá mundo</p>

Esta é uma tag HTML!! Neste exemplo, a tag <p> é tag de abertura, </p> a tag de fechamento, "Olá mundo" o conteúdo da tag e todos eles compondo um elemento.

Tags podem ter atributos

<img src"ninjas-unicórnios.png">

Em <img, o atributo src"..." aponta para a URL do arquivo

Não deve haver espaço entre um nome e seu valor.

Estrutura de um arquivo HTML

A tag <HTML>

Um dos três elementos principais que compõem a estrutura de um arquivo HTML é a tag <HTML>. Nela estão envolvidas todas as tags exceto a tag especial <!DOCTYPE html>

Dentro dela, deve existir as seguintes tags nesta ordem:

  • <head>...</head>
  • <body>...</body>

A tag <head>

Do inglês: cabeça

Contém meta-informação sobre este arquivo html, por exemplo:

  • Título da página no topo do navegador (obrigatório)
  • Codificação (enconding) da página
  • Ícone
  • Inclusão de arquivos(.css, .js)

A tag <body>

Do inglês: cabeça

Contém todo o conteúdo da página: tags de

  • parágrafos <p>...</p>
  • títulos e subtítulos <h1>...</h1>
  • imagens <img>

Tags Universais

Seletor id

É possível definir um nome que identifique um único elemento da página

Todo elemento html pode ter um atributo id. Exemplos:

<img src"..." id="capa-mongol"> <h1 id="Título-principal">Morango aos pedaços</h1>

Não pode haver atributos id com nomes iguais. Para cada atributo id deve se ter um nome diferente.

Seletor class

A class é parecida com o id, porém ela é usada para estilizar dois ou mais elementos da mesma forma. Exemplos:

<h1 class"blue">Sabre de luz</h1> <h2 class"blue">Estrela da Morte</h1>

Neste exemplo, como o h1 e o h2 possuem a mesma class, eles serão estilizados da mesma maneira.

Como colocar um Ícone ?

Certamente, uma das maiores dúvidas da nossa galáxia mas com uma solução mais simples do que se irritar com Jar Jar Binks!

Para adcionar um ícone(favicon) à sua página HTML basta você utilizar a tag <link>.

<link rel="icon" href="imgs/url-da-imagem">

Como o ícone não faz parte das informações que serão exibidas na página, ele deve ficar dentro da tag <head> e geralmente abaixo da tag <title>.

Principais tags de contúdo e seus atributos

Tag de Paragráfo ( <p> )

Quebras de linha são feitas automaticamente.

Tags de Títulos e Subtítulos ( h1...h6 )

Podemos ter até seis níveis de títulos! Eles vão de h1 a h6, sendo h1 o título de primeiro nível, h2 o subtítulo de segundo nível e assim por diante.

Temos seis níveis de títulos (h1...h6) mas podemos usar as tags quantas vezes quisermos.

Tag de Imagem ( <img> )

Para exibir imagens:

<img src"ninjas-unicórnios.png">

O atributo src"..." (abreviação de source) é usado para apontar o endereço URL da imagem.

Uma URL pode ser interna ou externa.

URL interna

A URL interna aponta para algo da própia página/site e pode ser relativa ou absoluta.

  • janelas-quebradas.html absoluto
  • imagens/patinhos.jpg absoluto
  • .../imgs/logo.png absoluto
  • /arquivos/sabredeluz.zip relativo

URL externa

Aponta para algo ( imagem, etc ) de outro site. Geralmente começa com: <http:// ou <https://.

  • Google
  • Moodle cefet
  • Wiki Star Wars

Exemplo 1: Imagem e arquivo HTML na mesma pasta

Arquivo HTML está em:

/brigadeiro/monstro/maravilhoso.html

A imagem está em:

/brigadeiro/monstro/monster-doces.jpg

Podemos utilizar seu endereço absoluto. Ficaria assim:

<img src"/brigaderio/monstro/monster-doces.jpg">

Ou seu endereço relativo. Ficaria assim:

<img src"monster-doces.jpg">

Exemplo 2: Imagem uma pasta adentro

Utilizando o endereço absoluto ficaria assim:

<img src"/brigadeiro/monstro/img/monster-doces.jpg">

Endereço relativo:

<img src"img/monster-doces.jpg">

Exemplo 3: Imagem uma pasta acima

Endereço absoluto:

<img src"/brigadeiro/monster-doces.jpg">

Endereço relativo:

<img src"../monster-doces.jpg">

Exemplo 4: Imagem duas pastas acima

Endereço absoluto:

<img src"/monster-doces.jpg">

Endereço relativo:

<img src"../../monster-doces.jpg">

Como incluir css ?

Ei jovem Padawan, não se desespere! Incluir o código css é tão fácil que até o desprezível Jabba the Hutt conseguiria!

Para incluir o código css, basta você digitar link e apertar tab, depois coloque o nome de seu arquivo css. Ficaria assim:

<link rel="stylesheet" href="estilos.css">
